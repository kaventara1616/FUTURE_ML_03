# Resume Screening System

## Description
This project screens resumes based on a job description using Natural Language Processing (NLP).

## Features
- Reads resumes from a CSV file
- Reads a job description
- Calculates resume matching score using TF-IDF and Cosine Similarity
- Ranks candidates
- Displays missing skills

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
